HTML (HyperText Markup Language) is the standard language for creating web pages. It uses elements and tags to structure content like headings, paragraphs, images, and links that browsers display visually.
CSS (Cascading Style Sheets) is the language used to style web pages. It controls colors, fonts, layouts, and spacing — making plain HTML content visually attractive and responsive across different screen sizes.
JavaScript is a programming language that makes web pages interactive. It handles dynamic actions like button clicks, form validation, animations, and real-time updates — bringing life to static HTML and CSS pages.
Angular is a TypeScript-based frontend framework by Google. It helps build fast, scalable single-page applications using components, and modules — making large web app development structured, and efficient.
